# UK Online Retail Sales Analysis (2009–2011)

## Project Overview
This project presents an end-to-end sales analysis of a UK-based online retail company using Power BI. The objective is to transform raw transactional data into meaningful business insights related to revenue, customers, products, geography, and seasonality.

The dashboard is fully interactive, mobile-ready, and designed to support data-driven decision-making.

## Business Questions Addressed
-What is the overall sales performance between 2009 and 2011?

-Which countries contribute the most and least to revenue?

-Who are the top and bottom revenue-generating customers?

-Which products drive the highest and lowest revenue?

-How does revenue vary by season and year?

-What is the contribution of returning customers?

## Key Features
-KPI cards for total revenue, orders, customers, and products

-Top & bottom analysis for products, countries, and customers

-Year-wise and seasonal revenue trends

-Returning customer analysis

-Interactive slicers for dynamic exploration

-Mobile-optimized report layout

-Exportable PDF version of the report

## Tools & Technologies
-Power BI Desktop

-Power Query (data cleaning & transformation)

-DAX (measures and calculations)

-Excel (source dataset)

## Project Structure
-Power BI report (.pbix)

-Exported PDF of the dashboard

## Outcome
The project demonstrates strong skills in data modeling, DAX, dashboard design, and business storytelling using Power BI. It is built with a recruiter and stakeholder audience in mind, focusing on clarity, usability, and actionable insights.
